본 연구에서는 expert_consensus 및 vote 라벨을 모델 학습에는 사용하지 않고 분석 결과 해석 및 검증 단계에서 참고 지표로 활용. 또한 데이터에는 센서 오류, 신호 포화 (saturation), 결측치 등 다양한 artifact가 포함되어 있으며, 이는 비지도 학습 모델이 실제 뇌파 패턴이 아닌 노이즈를 이상으로 탐지하는 문제를 유발. 따라서 데이터 정제 과정이 필수적으로 요구됨.

### 배경 및 필요성 

- **진단의 복잡성**

뇌파(EEG) 데이터는 고차원의 다채널 시계열 신호로 구성되어 있어 명확한 기준 없이 이상 패턴을 식별하는 것은 매우 어려움.

- **기존 접근의 한계**

대부분의 기존 연구는 전문가 라벨을 기반으로 한 지도 학습에 의존하며, 이는 새로운 유형의 이상 패턴을 발견하는 데 한계가 있음.

- **비지도 학습의 필요성**

라벨 없이 데이터 자체의 구조를 분석함으로써, 기존에 정의되지 않은 잠재적 이상 패턴 및 새로운 뇌 활동 유형을 발견할 수 있음

### 프로젝트 목표

**1. 잠재 구조 발견**

통계적 특징을 기반으로 EEG 데이터 내 숨겨진 패턴 및 군집 구조를 탐색

**2. 이상 패턴 탐지**

정상적인 EEG 패턴으로부터 벗어나는 비정상 신호를 자동 식별

**3. 해석 가능한 분석**

각 군집 및 이상 데이터의 통계적 특성을 분석하여 의학적으로 해석 가능한 결과 도출

**4. 경량 분석 시스템 구축**

복잡한 딥러닝 없이도 CPU 환경에서 동작 가능한 효율적인 EEG 분석 파이프라인 구축

### 주요 기능 및 특징

**1. 통계 기반 피처 엔지니어링 (Feature Engineering)**

각 채널별 시계열 데이터에서 통계적 특징을 추출하여 고차원 시계열을 저차원 feature 벡터로 변환. 초기에는 autocorrelation 기반 특징도 포함하였으나, 노이즈 민감성과 계산 비용 문제로 인해 제거하고 보다 안정적인 통계 기반 feature 중심으로 구성함.

**2. 비지도 학습 기반 패턴 분석**

**(1) 클러스터링**

- K-means / Gaussian Mixture Model
- EEG 패턴 자동 그룹화

→ 데이터가 스스로 어떤 구조로 나뉘는지 탐색

**(2) 이상 탐지**

- Isolation Forest / One-Class SVM

→ 정상 패턴에서 벗어나는 EEG 신호 탐지

✓ 비지도 이상 탐지 모델은 데이터 내에서 가장 분포적으로 이질적인 샘플을 탐지. 그러나 EEG 데이터에서는 이러한 접근이 실제 이상 신호뿐만 아니라 센서 오류 및 artifact를 함께 탐지할 수 있으므로 데이터 정제 과정과 병행하여 해석.

**3. 차원 축소 및 시각화**

- PCA / t-SNE / UMAP

→ 고차원 EEG 데이터를 2D/3D로 표현하여 패턴 구조 직관적으로 분석

**4. 사후 해석 (Post-hoc Analysis)**

- 각 클러스터의 특징 분석
- 전문가 라벨과의 비교

→ ex) 특정 클러스터가 seizure 패턴과 높은 연관성을 가지는지 분석

### 일정 및 계획

**1주차 | 데이터 탐색 및 전처리**

- EEG 데이터 로드 및 결측치 처리
- 채널별 신호 시각화 및 기초 통계 분석

**2주차 | 피처 엔지니어링**

- 통계적 특징 추출 함수 구현
- Feature 벡터 구성
- 데이터 내 artifact 및 노이즈 패턴 분석
- 비정상 신호 제거를 위한 필터링 기준 설정

**3주차 | 비지도 모델 적용**

- 클러스터링 알고리즘 적용 및 비교
- 이상 탐지 모델 구현

**4주차 | 결과 분석 및 해석**

- 군집 결과 시각화
- 라벨과의 관계 분석
- 이상 데이터 사례 분석 및 보고서 작성
- 이상 탐지 결과의 신뢰성 검증 (artifact vs 실제 EEG 패턴 구분)

### 기대효과

- **데이터 기반 패턴 발견**

기존 라벨에 의존하지 않고 EEG 데이터의 새로운 구조 및 이상 패턴 발견 가능

- **조기 이상 탐지**

명확한 진단 이전 단계에서도 이상 신호를 탐지하여 초기 경고 시스템으로 활용 가능

- **해석 가능성**

통계적 특징 기반 분석을 통해 의료진이 이해 가능한 형태의 결과 제공

### 기존 프로젝트의 문제점 개선

- **라벨 의존성 및 일반화 한계 (Label Dependency & Generalization)**

참고 프로젝트는 전문가 라벨(expert_consensus)을 기반으로 학습되는 지도 학습 방식으로, 학습 데이터에 존재하는 클래스에 대해서만 예측이 가능하다는 한계가 있음. 즉, 새로운 유형의 뇌파 이상 패턴이나 기존에 정의되지 않은 신호에 대해서는 적절한 대응이 어려움. 본 프로젝트는 비지도 학습을 통해 라벨 없이 데이터의 구조를 학습하므로, 기존에 정의되지 않은 잠재적 이상 패턴까지 탐지 가능함.

- **분류 중심 접근의 한계 (Classification Limitation)**

기존 방식은 EEG 데이터를 특정 클래스(Seizure, GPD 등)로 분류하는 데 초점을 맞추기 때문에, “정상과 얼마나 다른지” 또는 “이상 정도의 연속적인 변화”를 표현하기 어려움. 본 프로젝트는 이상 점수(anomaly score)를 기반으로 데이터의 연속적인 위험도 및 변화 정도를 정량적으로 표현할 수 있음.

### 기존 프로젝트의 성능 개선 (기술적 차이)

- **이상 탐지 성능 및 확장성 (Anomaly Detection Capability)**

기존 프로젝트는 주어진 6개 클래스 내에서의 분류 성능에 최적화되어 있음. 반면 본 프로젝트는 정상 패턴을 기준으로 데이터 분포를 학습하고, 그로부터 벗어나는 데이터를 탐지하는 구조로, 새로운 유형의 이상 EEG 패턴이나 라벨이 존재하지 않는 데이터에서도 확장 가능한 탐지 성능을 제공함.

- **추론 속도 및 자원 효율성 (Latency & Resource Efficiency)**

스펙트로그램 변환 및 딥러닝 모델을 사용하는 기존 방식 대비, 통계 피처 기반 접근은 단순 산술 연산으로 구성되어 있어 연산 복잡도가 크게 감소함. CPU 환경에서도 밀리초(ms) 단위 추론 가능하고, 실시간 스트리밍 EEG 분석에 적합.

### 결론

본 연구는 기존의 “정답을 맞추는 분류 문제”에서 벗어나, 라벨 없이 EEG 데이터 자체의 구조를 이해하고 잠재적인 이상 패턴을 발견하는 것을 목표로 함. 이를 통해 보다 유연하고 확장 가능한 EEG 분석 시스템 제안.

### 실제 환경 개선 및 반영 제안

기존 프로젝트들이 주로 병원 내 의료진의 판독 보조를 위한 정답 기반 진단 모델에 집중했다면, 본 프로젝트는 라벨 없이 EEG 데이터의 통계적 구조를 분석하고, 정상적인 신호 분포에서 벗어나는 패턴을 탐지하는 데 초점을 두고 있음.

다만 본 연구는 실제 의료기기 또는 실시간 EEG 장비와 직접 연동되는 시스템을 구현한 것은 아니며, 저장된 EEG 데이터를 기반으로 비지도 이상 탐지 구조의 가능성을 검토하는 단계에 해당. 따라서 본 프로젝트의 결과는 즉각적인 진단 도구라기보다, 향후 실시간 모니터링 시스템으로 확장하기 위한 기초 분석 및 프로토타입 설계로 해석할 수 있음.

- **이상 신호 조기 감지를 위한 확장 가능성 (Early Warning System)**

본 프로젝트에서 사용한 통계 기반 피처 추출과 Isolation Forest 모델은 새로운 EEG 샘플이 입력되었을 때 해당 신호가 기존 정상 분포와 얼마나 다른지를 anomaly score 형태로 계산할 수 있음. 이를 활용하면 향후 실시간 또는 준실시간 환경에서 EEG 신호를 일정 구간 단위로 나누고, 각 구간의 이상 점수를 계산하여 위험도를 판단하는 구조로 확장할 수 있음.

예를 들어 일정 시간 동안 수집된 EEG 구간에 대해 동일한 피처 추출 과정을 적용한 뒤, 학습된 scaler와 이상 탐지 모델을 사용하여 anomaly score를 산출. 이후 clean dataset에서 관찰된 score 분포를 기준으로 threshold를 설정하면 다음과 같은 단계적 경고 체계를 설계할 수 있음.

- Normal: 기존 EEG 패턴과 유사한 범위
- Caution: 정상 분포에서 다소 벗어난 구간
- Warning: 이상 점수가 낮아 추가 확인이 필요한 구간

이러한 방식은 발작 여부를 직접 진단하는 것이 아니라, 평소와 다른 신호 변화가 나타나는지를 조기에 감지하여 사용자 또는 보호자가 추가적인 확인을 할 수 있도록 돕는 보조적 구조로 활용될 수 있음.

- **재택 기반 이상 모니터링 시스템 (Home-based Anomaly Monitoring)**

휴대용 EEG 기기와 연계할 경우 본 프로젝트의 분석 파이프라인은 가정 환경에서 수집된 EEG 데이터를 주기적으로 분석하는 모니터링 시스템으로 확장될 수 있음. 사용자는 스마트폰이나 태블릿을 통해 현재 EEG 구간의 이상 점수 또는 위험 단계를 확인하고, 일정 기준 이상으로 비정상 패턴이 반복될 경우 병원 방문이나 전문의 상담을 고려할 수 있음.

그러나 실제 재택 모니터링 시스템으로 활용하기 위해서는 추가적인 검증이 필요. 특히 개인별 EEG 패턴 차이, 기기별 신호 품질 차이, 실시간 데이터 수집 과정에서 발생하는 artifact, 임상적 유효성 검증 등이 반드시 고려되어야 함. 따라서 본 프로젝트는 완성된 의료 서비스가 아니라, 비지도 학습 기반 EEG 이상 탐지의 가능성과 향후 시스템 확장 방향을 제시하는 연구로 볼 수 있음.
